# Rockpapersessiorgame
A simple Rock-Paper-Scissors game in JavaScript where you play against the computer. Demonstrates loops, conditionals, and DOM interaction.
## Features
- Play against the computer.
- Random computer moves.
- Tracks wins, losses, and ties.
- Interactive gameplay using prompts or buttons (depending on your implementation).

## How to Play
1. Open the game in a browser (if using HTML/JS)
2. Choose your move: `rock`, `paper`, or `scissors`.
3. The computer selects its move randomly.
4. The winner of each round is displayed.

## Skills Learned
- JavaScript basics: variables, conditionals, loops
- Random number generation
- User input handling
- DOM manipulation (if using a web interface)
